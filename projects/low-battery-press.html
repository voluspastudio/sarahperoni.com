01 ▸ overview.html
01 · overview

Low Battery Press.

A desktop diary for low-signal days.

Low Battery Press is a journaling system for people who are too tired to journal traditionally. The project reimagines journaling as a soft 90s-inspired desktop where users can type, talk, draw, save fragments, log music, and print emotional records later.

Instead of one blank page, the system gives users a personal desktop of tools. Each window captures a different kind of entry, from full journal posts to webcam check-ins, mood objects, songs, and printed archive pieces.

~ deliverables
  • Brand identity
  • Desktop interface
  • Journaling tools
  • Mood interaction system
  • Music log
  • Print archive
  • Icon system
02 ▸ desktop_system.os
02 · desktop system

The Desktop
as Journal.

The interface metaphor: a soft personal computer instead of a blank page.

The main interface is built like an old personal computer desktop. Icons act as entry points into different journaling tools: Journal.exe, Video Log, Notes.txt, Draw Pad, Mood Meter, Music Memory, Archive, and Print Dump.

The desktop format makes the experience feel personal and low-pressure. Users do not have to start with a perfect entry. They can open a window, drop in a thought, attach a mood, save a song, or print the log later.

~ system notes
  • Icons function as the primary navigation
  • Windows separate different types of emotional input
  • File names, folders, popups, and status bars create the 90s desktop logic
  • The interface feels playful without becoming a game
03 ▸ core_tools/
03 · core tools

Tools for different
kinds of entries.

Low Battery Press does not assume every thought becomes a paragraph.

Journal.exe

Type the words.

A writing window for typed entries and prompts. The cursor blinks. The save button glows.

Video Log

Talk for thirty seconds.

A webcam-style check-in for talking through the moment. Pauses, repeated words, and tone become a tiny written log.

Notes.txt

Loose thoughts.

A place for fragments, lists, and unfinished thoughts. Nothing has to become a full entry.

Draw Pad

Scribble it out.

A small sketch space for marks, symbols, and rough feelings. Save the drawing as an icon or a cursor.

Paint

Make a mess.

A playful visual tool inspired by early desktop paint programs. Pencils, brushes, fills, and chunky pixels.

The goal is to lower the barrier to documenting a moment without making the user perform a polished version of themselves.

04 ▸ mood_objects.app
04 · mood objects

Emoji as
interface.

Mood objects are small pixel icons attached to each post. Interactive, not static.

Users can add mood objects manually, or the system can suggest them based on the entry. When tapped, each object creates a small reaction inside the window: tears fall from sad icons, suns or sparkles drop from happy ones, hearts float upward, and error icons create a brief glitch.

~ object behaviors
  • Sad icons release pixel tears
  • Happy icons release suns or sparkles
  • Heart icons float upward
  • Error icons briefly glitch the post
  • Music icons connect the entry to a song
  • Each object becomes part of the printed archive

The print archive gives the digital journal a physical afterlife. Instead of disappearing into a feed, entries become small paper records that can be collected, clipped, folded, or saved.

~ print outputs
  • Receipt Log: one entry printed as a narrow strip
  • Mood Card: a compact card with battery level, mood objects, and entry text
  • Sticker Sheet: icons generated from repeated emotions
  • Monthly Dump: a zine-style recap of saved entries, songs, and mood patterns
  • Emotion Report: a printed grid of the user's emotional range over time
